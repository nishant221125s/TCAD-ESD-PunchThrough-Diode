# TCAD Simulation of Si/SiGe NPN Punch-Through Diode for ESD Protection

## Overview
This project investigates the behavior of NPN punch-through diodes using TCAD simulations for ESD protection applications. The study focuses on how device parameters such as doping, base width, and material composition (Si vs SiGe) affect electrical characteristics.

## Tools & Technologies
- Sentaurus TCAD
- Semiconductor device physics
- Python/plotting (if used)

## Key Contributions
- Simulated NPN punch-through diode structures for ESD applications
- Analyzed impact of base width (50–70 nm) and doping (1e16–1e18 cm⁻³)
- Compared Si and SiGe devices; observed improved conduction due to bandgap narrowing
- Studied punch-through mechanism enabling controlled conduction at high reverse bias
- Optimized parameters for electric field control and device performance

## Results
- Higher doping increases turn-on voltage and electric field peak
- SiGe improves carrier transport and reduces effective bandgap
- Punch-through enables fast transient conduction suitable for ESD protection
